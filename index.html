How to config the IDEA bundle maven

  1. the bundle maven home is: $IDEA_HOME/plugins/maven/lib/maven3, aka $MAVEN_HOME
  2. the config file is $MAVEN_HOME/conf/settings.xml

how to assemble a computer

Posted on 2021-01-22

装机

推荐两个我在B站上看的装机教程

干货，无废话，28分钟：https://www.bilibili.com/video/BV1jE411e7hw?p=1

冗长但幽默，装机+听相声，79分钟：https://www.bilibili.com/video/BV1qs41177LM

对于第一个简短的版本，它的分P已经很清楚了，不需要我做什么指引。

对于第二个详细版本，干货其实并不会更多，只是更多的时间在听相声，对于一些需要慢节奏否则听不懂的同学来说，是很好的选择。下面我将给出这个教程的笔记和导航：

推荐1.25倍速，既能听相声，又能尽快看完。

演示的配置：

  1. CPU：AMD 锐龙 2700K
  2. 主板：微星 X470 GAMING PLUS
  3. 内存：金士顿
  4. 显卡：七彩虹
  5. SSD：三星
  6. 机械：希捷
  7. 电源：酷冷至尊
  8. 散热：酷冷至尊
  9. 机箱：海盗船

合计16000。较为复杂，所以一般人在装自己用的电脑的时候，只会更简单。

03:23

开始硬件开箱。CPU的盒装和散装。

开始讲相声。各种相声只为一个目的：散片是正规的。散片比盒装便宜，也能买。

20:12

相声结束。继续拆包装。CPU尽量不要摔着了。可能会有损坏。

27:58

开始讲主板

具体怎么选，请看以前的视频。

CPU装入主板：根据CPU大小找到主板对应位置。抬起小铁杆，注意反正。CPU上的小三角与主板上的小三角对齐。然后放下小铁杆。CPU就装入主板完成啦。

36:30

开始散热器。

也是装入主板，根据螺丝数量和大小找到散热器的位置。把主板上的螺丝取下来，把散热器上的螺丝拧进去。小技巧：对角顺序去拧螺丝，不要完成一个再进行另一个，而是慢慢的一起拧。

49:53

开始内存条

金士顿的稳定性和兼容性还是ok的。

四条？安装内存是比较难的。

有金边的一条叫金手指。

最长的东西装在最长的位置。

先把主板上的机关打下来。主板上有四个槽，可能会有优先级，如果你只插入一个两个内存条，尤其需要注意这个优先级。

然后就能把豁口对准插进去，然后就能固定住了，四个都是如此。

01:07:48

开始固态硬盘。

这是M2接口的演示。

同样的找主板上的位置。斜着插入然后用螺丝固定。

01:18:53

结束主板上的安装。其他

introduction to tennis

Posted on 2021-01-21

网球入门教程

资料：https://www.bilibili.com/video/BV1hp4y1C7V3

适用人群：完全不懂网球的小白

我于2021.1.21看完这个教程，比较喜欢，教练手把手教你，一步一步分解，每个标准动作都讲得很清楚，还讲了如何练习的方法，让我眼睛已经看会了。至于身体会不会，现在正处于练习过程中。以下的时间戳是该视频讲到这个知识点的时刻。如果你想配合本笔记与视频一起观看，可以这样做：1. 用电脑网页端打开该链接。2. 点击视频右下角记笔记按钮。3. 复制进去。4. 点击时间戳可以跳转，当遇到教练展示标准动作的时候，可以点击网页笔记的拍照按钮截取图片保存到笔记本里面。

03:17

热身运动：头部，手掌，肩部，转腰，交叉双腿压腿，大腿弓步压腿。

05:19

正手握拍。三种，区别是击球点不同。

10:21

反手握拍。三种，一种用于切球，网前拦截。一种用于单手反拍抽球。一种是双手。

13:49

正手底线抽球。

侧身，球拍头部指向后场，球拍根部指向前场，击球点在身体侧前面，球拍扛在肩上。

适应运动。打球不要动手腕。球拍是推球的感觉。

22:60

准备动作，侧身拉拍，上步，击球，准备动作。

准备动作：双脚站立1.5倍肩宽。

24:03

双手反手抽球。右手正常正手，左手也正手但是拿 的地方更高一点。右手是辅助，用左手发力，动作一定记得和正手一样必须抬高，虽然可能会很别扭，但是就是要这样。

击球过程和正手一样。

31:09

单手反手抽球。偏向进攻性。姿势好帅有点大鹏展翅的感觉。

收尾的动作是指向前上方而不是肩上。

动作有点像武侠佩刀抽刀上去。

左手可以辅助一点点握拍。

37:26

单手反手削球。

和抽球动作类似，不过方向不一样，是从上往前下方走的。

要搭配转腰的动作，而不是仅仅靠手。

42:35

正反手击球秘诀

站位技巧：1.5倍肩宽，膝盖微微弯曲。

移动：先引拍，再跑位。跑出去，跳回来。

45:17

击球方向的处理。

就是基本动作的变形。击球方向与拍面和站位有着巨大的关系。

同样的击球动作，不同的击球时间能导致不同的拍面。提前击球打左前方，延缓击球打右前方。基本动作就打正前方。

50:56

力量与旋转。

力量导致速度，旋转导致球轨迹不规则。

力量：用拍头部位击球，（角速度一样，但是线速度更大）

旋转：用上拍边击球

54:53

放小球。

反手削球的动作，但是轻一点，高一点。而长削球就是比较直，比较远。

58:10

正手截击。

前面都是底线击球，这里开始进攻性的网前击球。

重心在前脚，拍在右前方击球，45度下角度向前推出去。

类似于砍刀或者锤子的动作。

当球来得比较高，想打得更低的时候怎么办？击球点变低，角度变小。击球点变高，角度更大甚至平行推，但是尽量不要往下压。

01:07:48

反手截击。

几乎一样和正手截击，只是换了方向。

如果反手不够力量，可以用双手。同样的拉拍动作，左手也发力。

01:15:12

截击秘诀。

  1. 膝盖要弯曲
  2. 来的球速越快，拦截幅度越小。
  3. 右边用正手，中间和左边来球用反手。
  4. 走位：侧身前移上步可以增加力量。平移用来找球。

01:19:29

发球基础。

  1. 站位：底线，离中线半米的位置。发球之前，脚不能超越中线和底线。
  2. 步骤：第一挥拍，第二抛球，第三击打
  3. 规则：发球需要发到对角场区。
  4. 从在肩上扛着，打到左下角。
  5. 发球热身，类似投掷手榴弹的动作。
  6. 抛球：在一点钟的位置抛球，抛球点较低，手肘必须伸直，不要旋转。落点应该在左脚右前方二十公分位置。
  7. 可以先用手练习，可以从前往后练习，先从中线发球练起。然后慢慢退到底线，力量会需要稍微加大。
  8. 身体：向右侧身，然后转到左边，脚基本不要动。右腿关节不要向下压。
  9. 收拍：可以用左手接住，不要因为用力过猛而达到自己腿，也可以用惯性在左腿外侧滑动。千万不要打到腿，很痛的。
  10. 处理下网和出界：下网是因为抛球比较前。出界是因为抛球抛后了。

01:39:26

高压球技术

对手打出高球，我们可以进行高压球的进攻。动作和发球动作差不多。

不同的是：可以用左手抬起来，辅助判断球的位置。

如何阅读营养成分表？

我的参考资料

https://www.zhihu.com/question/22632481/answer/335132267

我的总结

  1. 三列，第一列是名字，第二列是含量，注意单位，第三列是含量占人体每天所需的比例。根据这些数据可以估算吃这个食品会摄入什么东西。
  2. 国家规定：营养成分表必须带有能量，碳水化合物，脂肪，蛋白质，钠。这些都是对健康很重要的元素，尽量平衡摄入。过少摄入可能影响健康，过多摄入可能导致肥胖高血压。
  3. 也可以适当看看配料表。国家规定：配料表的排列顺序必须按照用量的大小来排列。可以估计这份食品的原料大概是什么。
  4. 关于单位。我国一般使用千焦为单位。很多地方也使用千卡为单位。换算关系为1kcal=4.184kj。

how to install Linux on your computer

why need Linux?

  1. My computer is out of date, and it performs not very well with the windows10 OS.
  2. Although my computer performs well with windows7, MS do not support win7 any more, and win7 do not support OneDrive very well.
  3. The software that only work on windows do not bother me.
  4. Linux is good for Development.

why manjaro

  1. manjaro xfce is tiny and will not consume lots of resources.
  2. I like it.

how to install

just follow the official tutorial: https://manjaro.org/support/firststeps/

or you wanna refer to other’s tutorial: https://segmentfault.com/a/1190000022575018

how to use it

  1. select English to be your default language
  2. no need to install MS office, just use the online version, including OneDrive
  3. install daily software like Chrome, music, Typora
  4. install development environment: GCC, python, java, go, vscode, Jetbrains, Database, git
  5. choose your desktop wallpaper: full dark will be ok

Do not do this in algorithm tests

Posted on 2020-09-05 Edited on 2021-01-21

很多的笔试都是在牛客上考试的，但是我一直是在LeetCode上面刷题，今天发现一个重大问题。

LeetCode上面ListNode的定义是不需要你自己去再重新写一遍的，但是牛客上面要，所以这就导致一个问题：即使你的代码正确，会报错ListNode未定义！！！！！！！！！然后就case通过率0

难怪我之前为什么有些问题，明明写得还行，就是一个case都过不了，而且考试模式下面不会跳任何错误提示。所以根本不可能知道这个问题！！！

以后刷题，必须牛客和LeetCode兼得，不能让这种细节伤害了成绩。
